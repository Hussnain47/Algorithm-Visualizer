Quick Sort

Quick Sort is recursive sorting algorithm. In which we pick an item as pivot and put the items larger than it on the right and smaller on left.And run on each side again till the list is sorted

Psuedo Code :

To sort a given Array 'A':

  1. We will pick the last item as pivot and partiion the item by using two pointers where one is start part(i) of the partioning list and the other is the end(j) of it. We will a outer loop that will run from i to j.
  2. Then we will run inner loop where we will increase i pointer if till the item is less than the pivot and do the same with j but we will decrease j pointer.
  3. Now we have found items that are on the wrong side of the pivot and we will swap them till all the elements less than pivot are on the right of the pivot and all the items greater are on the left.
  4. Then we will run the quick sort on the left part of the pivot and on the right part of the pivot. and it will result to a sorted list.

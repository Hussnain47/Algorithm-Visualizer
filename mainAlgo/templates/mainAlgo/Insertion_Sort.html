Insertion Sort

Insertion sort iterates through the input list, checking one element each time, and producing a sorted output list. Insertion sort takes one element from the input array at a time, identifies where it belongs in the sorted list, and inserts it there. It keeps on until there are no more input items.

Psuedo Code :

To sort a given Array 'A':

  1. We will iterate through the list from arr[0] to arr[n-1].
  2. Then in each iterations we will save the current item.
  3. We will compare this item to all its predecessor. And if the predecessor is greater then we will move the items one position up till we reach the item which is less than the current item or end of the list and put the current item there.

Binary Search Tree

BST is an Abstarct Data Type.The items are stored the form of tree where the items less than the root are on left and greater on right.

Psuedo Code :

Inserting:

  1. We will start from the root.
  2. Iterate till the root is null. And move toward left if the item is less than the given item. or right if it is greater. and storing the item. This will be the parent of the node to be inserted
  3. Now, If the given item is greater than it will be the right of the parent node else it will be the right.

Searching:

  1. We will start from the root.
  2. Iterate till the root is null. And move toward left if the item is less than the given item. or right if it is greater. and if it is equal to the given item returning true.else returning false.

Deleting:

  1. We will start from the root.
  2. Iterate till the root is null. And move toward left if the item is less than the given item. or right if it is greater. till we find the node.
  3. If the node doesn't have any children. Then just delete it.
  4. If the node doesn't have a children on left then removing the current node and let the right take its place. and vice versa.
  5. If the node does have node on left and right then finding the successor of current node and swapping the their places and deleting the given node.

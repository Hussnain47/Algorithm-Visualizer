Bubble Sort

Bubble sort is a basic sorting algorithm that iterates through a list, comparing nearby entries and swapping them if they are out of order. The list is sent through again and again until it is sorted.

Psuedo Code :

To sort a given Array 'A':

  1. We will iterate through the list from arr[0] to arr[n-1].
  2. Then in each iterations we will swap the items if they are out of order

Merge Sort

Merge Sort is recursive sorting algorithm. which divides the given array into two recursively and sorts them when merging each part.Till the whole list is sorted

Psuedo Code :

To sort a given Array 'A':

  1. We will divide the array into two parts And calling Mergesort function on each half .If the size of the list is one then returning it
  2. Then merging it. To merge we will check each items of the list and put them in the list accordingly.
  3. Return the merged list till the list is sorted.
